# Glossário

Termos jurídicos e técnicos usados no projeto.

## Termos jurídicos

| Termo | Definição |
|-------|-----------|
| **Admissibilidade** | Exame preliminar que verifica se um recurso preenche os requisitos formais para ser julgado pelo tribunal superior |
| **Recurso Especial (REsp)** | Recurso ao STJ contra decisão que viola lei federal (art. 105, III, CF) |
| **Recurso Extraordinário (RE)** | Recurso ao STF contra decisão que viola a Constituição Federal (art. 102, III, CF) |
| **Recorrente** | Parte que interpõe o recurso |
| **Recorrido** | Parte contrária ao recurso |
| **Acórdão** | Decisão colegiada de um tribunal |
| **Permissivo constitucional** | A alínea do artigo constitucional que fundamenta o recurso (ex: art. 105, III, "a") |
| **Óbice** | Impedimento que barra a admissão do recurso (ex: falta de prequestionamento) |
| **Prequestionamento** | Exigência de que a matéria tenha sido discutida no tribunal de origem |
| **Dissídio jurisprudencial** | Divergência de interpretação entre tribunais sobre a mesma lei |
| **TJPR** | Tribunal de Justiça do Estado do Paraná |
| **STJ** | Superior Tribunal de Justiça |
| **STF** | Supremo Tribunal Federal |
| **Minuta** | Rascunho de decisão judicial |

## Termos técnicos

| Termo | Definição |
|-------|-----------|
| **Pipeline** | Sequência de processamento em 3 etapas |
| **Chunking** | Divisão de texto longo em partes menores para processamento |
| **Checkpoint** | Estado salvo em disco para recuperação em caso de falha |
| **Token** | Unidade de texto usada pela API da OpenAI para cálculo de custo |
| **Temperature** | Parâmetro que controla a "criatividade" da IA (0.1 = conservador) |
| **Hallucination** | Quando a IA inventa informações que não existem nos documentos |
